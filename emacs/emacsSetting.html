company-mode配置

(require 'company) (setq company-idle-delay .3) ; decrease delay before autocompletion popup shows (setq company-begin-commands '(self-insert-command)) ; start autocompletion only after typing (define-key company-mode-map (kbd "C--") 'company-filter-candidates) (setq company-tooltip-margin 2)

Fill-Column-Indicator插件 一根80行的基准线

(add-to-list 'load-path "c:/emacs/.emacs.d/Fill-Column-Indicator/")
(require 'fill-column-indicator) ;; run M-x fci
(setq fci-rule-column 80)
(setq fci-rule-width 1)
(setq fci-rule-color "darkorchid")

防止页面滚动时跳动， scroll-margin 3 可以在靠近屏幕边沿3行时就开始滚动，可以很好的看到上下文。

(setq scroll-margin 3 scroll-conservatively 10000)
